# Nutrillent
Originally named "GLED Nutrient Tracker" and created as a prototype for the online virtual hackathon "Hack the Pulse 2022".
https://devpost.com/software/hackthepulse
Collaborators:
Grace G.
Emma D.
Dora C.

After the hackathon ended, I finished programming the web app on my own and renamed it to "Nutrillent". As my skills improve, I hope to continue working on this project and continue to make it better. The next feature that I would like to implement is an editable table that can allow users to visually keep track of the food they are eating, along with its nutritional information. I think that this could also be a good opportunity for me to learn how to use React. Additionally, I plan on making the website more aesthetically pleasing and responsive with ReactJS and Bootstrap.

Nutritional information was retrieved from the US Department of Agriculture (USDA) FoodData Central API.

Deficiency data and daily nutritional intake recommendations were based on US Food and Drug Administration's (FDA) Reference Daily Intakes (RDIs).
https://www.fda.gov/media/99069/download
